# RAG Framework Chatbot

This project demonstrates how to build agent-based AI applications using modern LLM tools and vector databases.

---

## Environment Variables

OPENAI_API_KEY= XXXXX

ANTHROPIC_API_KEY=YYYYY


PINECONE_API_KEY = XXXXX

PINECONE_ENVIRONMENT = us-east-1


## Tech Stack

- OpenAI API – for language model capabilities  
- Anthropic Claude API – alternative LLM provider  
- Pinecone – vector database for embeddings and retrieval  
- LangChain – framework for building agentic workflows  

## To run

1) pip install -r .\requirements.txt
2) Streamlit run main.py
